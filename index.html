Brauchen Sie eine Website?

Ich bin ein Webdesigner/Web Entwickler aus Freiburg und ich erstelle Websites, die einfach zu bedienen sind, schnell laden, ohne CMS von jedem Gerät aus aktualisiert werden können und kostenlos gehostet werden.

Vorteile

  • Kostenloser Server

    Ihre Website wird kostenlos gehostet, das heißt Sie müssen nicht für einen Webserver sorgen oder für Hosting Ihre Website bezahlen.

  • Kostenloses SSL-Zertifikat

    Außerdem müssen Sie kein SSL-Zertifikat erwerben, um HTTPS zu aktivieren, da dieses kostenlos für Sie bereitgestellt wird.

  • Inhaltsaktualisierung von überall

    Ihre Website-Inhalte werden auf GitHub gespeichert. GitHub gehört zu Microsoft (ein US Unternehemen) der schützt Ihre Dateien durch Versionskontrollsystem. Durch GitHub können Sie Ihre Website von jedem Gerät sogar einem Smartphone aktualisieren.

  • Hochleistung

    Eine Leistungsbewertung von 90-100, gemessen mit PageSpeed Insights von Google.

  • Barrierefreiheit gemäß EU-Richtlinien

    Ihre Website wird auf Konformität mit WCAG 2.1 getestet, um sicherzustellen, dass sie die Barrierefreiheitsanforderungen erfüllt, die in vielen Ländern, einschließlich der Europäischen Union, durchgesetzt werden.

  • Konformität mit der DSGVO (GDPR)

FAQ

  • Wieviel kostet meine Website?

    Die Websites, die ich erstelle, kosten zwischen 1.500 und 10.000 Euro, abhängig von den individuellen Anforderungen eines Projekts. Wir besprechen im Vorfeld alle Möglichkeiten, um unnötige Kosten zu vermeiden.

  • Was sind die Voraussetzungen, um Ihren Service nutzen zu können?

    Bevor wir eine Website erstellen und live schalten, müssen Sie die folgenden Schritte ausführen:

    1. Ein kostenloses Konto auf GitHub registerien. Ihr Website-Code als auch Ihre Inhalte werden in einem Repository auf GitHub gespeichert. GitHub ist ein Onlinedienst zur Versionsverwaltung für Software-Entwicklungsprojekte. Das Unternehmen gehört zu Microsoft und hat seinen Sitz in den USA. Ich kann Ihnen bei Bedarf bei der Einrichtung Ihrem GitHub Konto helfen.
    2. Erwerben Sie einen Domainnamen für Ihre Website. Wenn Sie keine Domain besitzen, können Sie eine Domain bei hover.com registrieren und ich kann Ihnen bei Bedarf bei der Einrichtung Ihrer Domain helfen.
    3. Bereiten Sie den Inhalt Ihrer Website vor

  • Muss ich technisch versiert sein, um meine Website zu aktualisieren und zu warten?

    Für die Aktualisierung und Pflege Ihrer Website sind keine technischen Kenntnisse erforderlich, außer grundlegender Vertrautheit mit einigen HTML-Tags, die Sie in weniger als einer Stunde erlernen können. Gegen eine geringe Gebühr können Sie die Wartung auch an mich auslagern.

  • Wie aktualisiere ich meine Website?

    Sie können von jedem Gerät wie einem Mobiltelefon über einen Browser auf Ihr GitHub Repository zugreifen. Jedes Mal, wenn Sie eine Änderung an Ihrem Repository vornehmen, stellt GitHub Ihre Änderungen automatisch auf Ihrer Live-Site bereit. Ein CMS oder andere komplexe Lösungen um Ihre Inhalte verwalten sind nicht erforderlich. Da sich Ihre Inhalte in einem Repository befinden, müssen Sie sich keine Sorgen darüber machen, Backups zu erstellen oder Ihre Website durcheinander zu bringen, da Sie unerwünschte Änderungen in einem Repository jederzeit problemlos rückgängig machen können.

  • Wie lange dauert es bis meine Website live geht?

    Der Prozess der Gestaltung und Entwicklung Ihrer Website dauert in der Regel mindestens einen Monat. Wenn Sie den Inhalt Ihrer Website bereits fertiggestellt haben, geht es schneller voran.

  • Gibt es eine Garantie?

    Ja, Ihre Website wird mit einer lebenslangen Garantie geliefert, dass sie in der neuesten Version von Google Chrome, Mozilla Firefox, Safari und Microsoft Edge auf allen Geräten, die diese Browser unterstützen, einwandfrei aussieht und funktioniert. Diese Garantie erlischt, wenn Sie oder jemand anderes den Code Ihrer Website manipuliert.
