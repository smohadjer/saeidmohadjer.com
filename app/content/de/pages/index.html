Vorteile

  • Kostenloser Server und Hosting über HTTPS

    Ihre Website wird kostenlos auf Github gehostet. Auf diese Weise werden nicht nur Ihr Website-Code und Ihre Inhalte durch ein Versionskontrollsystem geschützt, sondern Sie müssen auch keinen Webserver und Hosting Ihrer Website einrichten und bezahlen. Außerdem müssen Sie kein SSL-Zertifikat erwerben, um HTTPS zu aktivieren, da dieses kostenlos von Github bereitgestellt wird.

  • Aktualisieren Sie Ihre Website von überall

    Sie können Ihre Website von jedem Gerät wie Ihrem Smartphone oder Ihrem Computer und direkt im Browser aktualisieren. Die genauen Schritte werden mit Ihnen geübt und Sie erhalten Anweisungen, denen Sie selbstständig folgen können.

  • Hochleistung

    Ein Leistungswert von über 90%, gemessen mit PageSpeed ​​Insights von Google.

  • Barrierefreiheit

    Ihre Website wird auf Konformität mit WCAG 2.1 getestet, um sicherzustellen, dass sie die Barrierefreiheitsanforderungen erfüllt, die in vielen Ländern, einschließlich der Europäischen Union, durchgesetzt werden.

Projektablauf

Entdeckungsphase
Bevor ich ein Projekt beginne, würde ich versuchen, so viel wie möglich über Ihr Unternehmen und Ihre Bedürfnisse herauszufinden. Wir werden verschiedene Ideen sammeln, um herauszufinden, welche Lösung am besten zu Ihren Bedürfnissen und Zielen passt.
Inhaltsphase
Der Inhalt ist der König, also werden wir zusammenarbeiten, um Text und andere Elemente wie Fotos und Videos vorzubereiten. Inhalte fertig zu haben, auch wenn sie noch nicht im endgültigen Zustand sind, hilft uns, die Website um die Inhalte herum zu gestalten und nicht umgekehrt, sodass wir sicherstellen können, dass wir eine einzigartige Lösung entwickeln, die sich von Standardseiten abhebt, die mit automatisierten Online-Tools erstellt wurden.
Design-Phase
In dieser Phase erstelle ich das allgemeine Layout und Design und überarbeite es basierend auf dem Feedback, das ich von Ihnen erhalte. Accessibility und Usability spielen in dieser Phase eine große Rolle.
Entwicklungsphase
Nachdem das Design abgeschlossen ist, werde ich Seiten und Komponenten erstellen, die auf der Website verwendet werden.
Test- und Bereitstellungsphase
In dieser Phase testen wir die Lösung auf verschiedenen Geräten von Smartphones über Tablets bis hin zu Desktops und beheben jedes Problem, das wir finden. Sobald wir mit dem Testen fertig sind, verlinke ich Ihre Domain mit dem Github-Repository Ihrer Website, was bedeutet, dass Ihre Website live ist.
Wartungsphase
In dieser Phase werde ich dem Kunden beibringen, wie er seine Website aktualisiert, indem er Inhalte auf Github bearbeitet.

FAQ

  • Wieviel kostet das?

    Die meisten Websites, die ich erstelle, kosten zwischen 1000 und 5000 Euro, abhängig von den individuellen Anforderungen eines Projekts. Zum Beispiel, wenn Sie Fotoshootings oder benutzerdefinierte Animationen benötigen, die die Kosten erhöhen würden. Wir besprechen im Vorfeld alle Möglichkeiten, um unnötige Kosten zu vermeiden.

  • Wie aktualisiere ich meine Website?

    Ihre Website wird auf Github gehostet, d. h. sowohl Ihr Website-Code als auch Ihre Inhalte werden in einem Repository auf Github gespeichert. Sie können von jedem Gerät wie einem Mobiltelefon über einen Browser auf Ihr Repository zugreifen. Jedes Mal, wenn Sie eine Änderung an Ihrem Repository vornehmen, stellt Github Ihre Änderungen automatisch auf Ihrer Live-Site bereit. Ein CMS (Content Management System) oder andere komplexe Tools sind nicht erforderlich. Wählen Sie einfach eine Datei aus, nehmen Sie Ihre Änderungen vor, indem Sie sie bearbeiten, und speichern Sie sie dann.

  • Muss ich einen Domainnamen kaufen?

    Ja, Sie sind für die Registrierung Ihrer Domain und die Zahlung der anfallenden Gebühren verantwortlich. Wenn Sie noch keine Domain besitzen, können Sie eine Domain bei hover.com registrieren und ich kann Ihnen bei Bedarf bei der Einrichtung Ihrer Domain helfen.

  • Wie lange dauert es, bis meine Website live geht?

    Der Prozess der Gestaltung und Entwicklung Ihrer Website dauert in der Regel etwa einen Monat. Wenn Sie den Inhalt Ihrer Website bereits fertiggestellt haben, geht es schneller voran.

  • Gibt es eine Garantie?

    Ja, Ihre Website wird mit einer lebenslangen Garantie geliefert, dass sie in der neuesten Version von Google Chrome, Mozilla Firefox, Safari und Microsoft Edge auf allen Geräten, die diese Browser unterstützen, einwandfrei aussieht und funktioniert. Diese Garantie erlischt, wenn Sie oder jemand anderes den Code Ihrer Website manipuliert.
