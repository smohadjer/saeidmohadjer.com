Website für Slaven Gluscevic

Die Website nutzt GitHub und Vercel als Repository und Hosting-Plattform. Alle Backend-Funktionen werden über serverlose Node.js-Funktionen implementiert, die in eine MongoDB-Datenbank lesen und schreiben. Die Website ermöglicht es Benutzern, sich online für Tenniscamps und/oder Trainingskurse anzumelden. Die Daten werden über JavaScript als JSON aus der Datenbank abgerufen und mithilfe von Handlebars Template-Engine in HTML konvertiert.

Jahr: 2023
