Closing an open port via Mac's terminal

Mac

Using following command in terminal you cana get a list of all open ports:

netstat -an | grep LISTEN

To get the process ID (PID) of port you wish to close, use 

lsof -i :PORT-NUMBER
. You may need to use sudo.

lsof -i :3000
COMMAND   PID USER   FD   TYPE             DEVICE SIZE/OFF NODE NAME
node    49835   sm   22u  IPv6 0xc947e947a81c2535      0t0  TCP *:hbci (LISTEN)

Then use the PID to kill the process listening on your port:

sudo kill -9 PID
