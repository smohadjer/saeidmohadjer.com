Vercel Hello World Example

Vercel Node.js 
mkdir test
cd test
mkdir public api
cd api
touch index.js
cd ..
cd public
touch index.html
cd ..

Open index.html in your editor and paste the following snippet in it:



    
        
        Hello World
        
    
    
        
Hello World

Open index.js in your editor and paste the following snippet in it:

export default function handler(req, res) {
    res.setHeader('Content-Type', 'text/plain')
    res.write('Hello World')
    res.end()
}

In ycli/terminal run 

vercel dev
and select default answer for all questions. Vercel cli will create the test project under your existing Vercel account and start a local server at: http://localhost:3000/ where you can see your index.html and your api can be invoked at: http://localhost:3000/api/

In cli/terminal run 

vercel deploy
to deploy your test project to production and make it publicly available on web.
