Do I need a server for my Website?

No, you don’t. Your website will be hosted on GitHub. GitHub offers free storage and hosting for public repositories. You can access your GitHub repository from any device meaning that you can even update your site directly from your cell phone. Every time you make a change on Github, it will automatically update your live site for you within a minute.

Can I host my website on my own server?

Yes, but then you need to contact your server's administrator to figure out how to update your website.

Do I need a domain name?

Yes, you are responsible for registering your domain name and paying annual fee (usually around 15€ per year) as well as pointing your domain to your GitHub page. Instructions for this will be provided to you via email and I can help you with this process if you need assistance.

Can I have a different design or functionality byond what is presented on demo website?

If you need a design that is considerably different from the demo website, it will incur additional costs to implement it, however minor changes in layout can be accomodated at no extra cost. Same is true for custom functionalities.

Can I update content of my Website?

Yes, you can update content of your website from any device directly in the browser on github.com. Your website code and content will be stored on a repository on GitHub. The changes you make there will go live within a minute using an automated deployment process. No need to learn to use a CMS or install anything.

How long does it take you to deliver my Website?

It usually takes 10 to 15 days to design, build and push your site live. If you have already prepared the content of your website (text and visual assets such as photos) the progress will be faster.
