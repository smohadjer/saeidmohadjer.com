Projects

  • Filterlist.js
    Filterlist is a small (4kb) TypeScript library with no dependencies that allows filtering of arbitrary HTML elements via data attributes. Filtering can be triggered using URL parameters, or by adding form elements to page, or programmatically via another script.
    Demo | GitHub | NPM
  • Tournament Oraganizer
    This web app stores and displays results of tournaments. It allows setup and selection of seasons, players' ranking and history, and individual tournament results. Data fetching is done by JavaScript from MongoDB by invoking Node.js API endpoints and then injecting fetched data into Handlebars templates to render views on client side. API authorization for post requests is implemented using JWT inside a middleware. App can run using Express server on a Node.js runtime as well as on Edge runtime via Serverless functions.
    Serverless Demo | GitHub
  • Website for Slaven Gluscevic
    Designed and built a Website that allows online registration for tennis training courses and camps. The admin section allows trainer to view registration data, write notes for students, update training schedule, and so on. Registration data is posted to serverless functions which then validate and write to a MongoDB instance as well as sending Email confirmation to registrants and trainer. Static content is stored on GitHub and auto-deployed to Vercel for live hosting.
