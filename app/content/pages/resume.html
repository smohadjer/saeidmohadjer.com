Projects

  • Website for Slaven Gluscevic
    Designed and built a Website that allows online registration for tennis training courses and camps. The admin section built as SPA allows the trainer to view registration data, write notes for students, update training time slots, and so on. Registration data is submitted to a serverless API which then validates and writes data to a MongoDB instance. All static content and assets are stored on GitHub and auto-deployed to Vercel for live hosting. I also wrote a frontend build that generates markup and compiles all resources for production as well as local testing.
    Visit Website
  • Tournament Oraganizer
    A web app that stores and displays results of tournaments. It allows setup and selection of seasons, players' ranking and history, and individual tournament results. App is built using MVC pattern where JavaScript fetches data from MongoDB by calling a serverless API and then injects data into Handlebars templates to render different views. API is written in Node.js and built using Vercel's serverless functions. Basic authentication for admin is implemented via a Vercel edge middleware.
