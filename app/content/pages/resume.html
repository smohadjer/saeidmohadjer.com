Personal Projects

  • Website for Tennis Trainer Slaven Gluscevic
    Designed and developed a Website for my club's tennis trainer to allow users register online for training courses and camps. The website features an admin section built as SPA that allows the trainer to view membership data, leave notes for students, update training time slots, and so on. Registration data is submitted to a serverless API (Node.js) which then validates and writes data to a MongoDB database. All static content and assets are stored on GitHub and auto-deployed to Vercel for live hosting. I wrote a frontend build that generates markup and compiles all resources for production as well as local testing.
    Visit Website
  • Tournament Oraganizer App:
    A web app for storing and displaying results of tournaments such as ranking of players, result of each tournament, player's history, and so on. The UI is built without any frameworks using HTML, CSS, and JavaScript only. Data is stored and fetched from a MongoDB database using Vercel's serverless Functions written in Node.js. JavaScript fetches data from API and renders HTML by populating Handlebars templates with data. Basic authentication for admin page is implemented via a Vercel Edge middleware.
